REQUIREMENTS
This launcher has a few requirements to work:
1 - windows
2 - java 17 or newer must be installed, the launcher will automatically find it on your device
3 - the feature of creating directories was not tested before, thereby might not work, which means:
- \AppData\.minecraft
- .minecraft\versions
- .minecraft\libraries
directories (folders) should be present, content not needed as the launcher is able to install versions by itself (tested)
